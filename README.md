# React + TypeScript + Vite

# Pasos de instalación

1. Clonar repositorio: `https://github.com/DanielZC/SPA.git`
2. Estar dentro de la carpeta del proyecto y ejecutar `npm install`
3. Ejecutar `npm run dev` para iniciar el servicio del front
4. Para acceder debe ingresar al link http://localhost:5173/
